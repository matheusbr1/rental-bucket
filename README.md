# Rental Bucket (in progress 🚧)

Application built for create, edit and list services, clients, trucks ands drives of some rental bucket company.

I'm bulding the API that this aplication uses too:
<a href='https://github.com/matheusbr1/rental-bucket-api' target='_blank' />Access API repo</a>

### For execute in your machine ☕

<pre>
On project folder

$ yarn install
$ yarn start
</pre>
